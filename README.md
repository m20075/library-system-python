# 📚 Library System (Python)

A simple command-line Library Management System built with Python.  
Manage your book collection easily — add, view, search, and delete books, with automatic data persistence using JSON.

## ✨ Features
- ➕ Add new books (title, author, year)
- 📖 View all books in the library
- 🔍 Search for a book by title
- 🗑️ Delete a book by title
- 🔢 Count total books
- 💾 Automatic saving/loading via JSON — your data is never lost

## 🚀 How to Run
```bash
python Untitled-1.py
```

## 🛠️ Built With
- Python 3
- `dataclasses`
- `json`
